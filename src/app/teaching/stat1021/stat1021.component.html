Lecture Description

This short lecture series provides an introduction to the daily practices and workflow for a data scientist/engineer in the industry, with an emphasis on building intuition on network data. Specifically, we will sample and analyze the in-class social network and introduce broadly applicable concepts in network analysis.

The lectures will be delivered as three integrated parts: on using Git and GitHub, on website hosting on GitHub Pages, and on a very short (but not hurried) introduction to Network Science. These will include (web) software development methods and tools; graph theory and graph concepts; community detection; the implications of popular concepts such as "six degrees of separation" and the "friendship paradox".

After these lectures, the students will be able to publish an open source software on GitHub that analyzes and visualizes network data and host a personal/professional website on GitHub Pages.

Prerequisites

The materials will be accessible to students with little programming experience; however, they must be able to write and run Python scripts on the command-line. This tutorial might also appeal to people being interested in Network Science. Yet it will focus on the practical side, i.e. using Python for network data analysis. Web development experience is not required but a prior exposure to HTML/CSS/JavaScript would be useful.
